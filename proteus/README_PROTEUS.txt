Instructions to open Proteus simulation:
1. Open Proteus 8 Professional.
2. Create a new project or open the provided .dsn file (if provided).
3. If no third-party model is required, use the built-in "GRAPHIC LCD 128X64" (ST7920) in Proteus.
   - Place the component: Mode: ST7920, configure for serial mode if available.
   - Connect P1.0 -> SCLK, P1.1 -> SDIN, P1.2 -> RS (as in code)
4. Connect VO to a 10k potentiometer (middle pin to VO), VCC to +5V, GND to GND.
5. Place AT89C51 and load the hex file generated by Keil (to be added).
6. Connect BEEP to P3.4 and a passive buzzer (or LED for testing).
7. Run simulation.

If your version of Proteus lacks a specific ST7920 model, use the "GRAPHIC LCD" generic module and enable serial mode or use a ready-made 128x64 module from Proteus library.
